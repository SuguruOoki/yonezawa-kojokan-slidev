# 概要

米沢興譲館高校で後輩相手にお話しする時に利用するスライドです。

#　ローカルでみるとき

## npm

- `npm install`
- `npm run dev`
- visit http://localhost:3030

## yarn

- `yarn install`
- `yarn dev`
- visit http://localhost:3030
